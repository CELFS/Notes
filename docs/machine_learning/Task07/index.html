7.3 贝叶斯分类器与贝叶斯学习​

  • ML 的流派
    • 从概率与统计的角度
      • 频率主义
        • 统计学习
        • SVM
      • 贝叶斯主义

    • 分布估计

  • 【感悟】

    • 以前乱学，把两者界线混淆了，殊不知是矛盾的观点。

    • 因此，并非用了贝叶斯公式就是贝叶斯学习，区别在于背后的分布类型，求得参数的方法。
